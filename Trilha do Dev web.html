Linguagens de Programação

Trilhas de imersão para Devs

    Estruturas web

  • 💰 HTML:

    • - Estrutura básica das páginas web.

  • 🔢 CSS:

    • - Estilização das páginas web. Estilize suas páginas e aprenda sobre layout, cores e fontes.

  • 🔮 JavaScript:

    • - Entenda a programação no lado do cliente para adicionar interatividade.

      Desenvolvimento Front-end

  • 🔢Frameworks de CSS:

    • - Aprenda Bootstrap ou Tailwind CSS.

  • 🔮Frameworks de JavaScript:

    • - Aprenda frameworks como React, Vue ou Angular.

      Desenvolvimento Back-end

  • 🏆Linguagens de Programação:

    • - Aprenda Node.js (JavaScript), Python (Django/Flask), Ruby (Rails) ou PHP.

  • 🌐Bancos de Dados:

    • - Conheça SQL (MySQL, PostgreSQL) e NoSQL (MongoDB).

      APIs de Integração

  • 🎬JSON:

    • - Formato de troca de dados, fundamental para APIs.

  • 🎬RESTful APIs:

    • - Aprenda a criar e consumir APIs.

      APIs de Implantação

  • 🃏Servidores:

    • - Apache / Handoop / XAMPP/ FTP / DNS / Shell

  • 🃏Implantação:

    • - Aprenda sobre serviços de hospedagem
